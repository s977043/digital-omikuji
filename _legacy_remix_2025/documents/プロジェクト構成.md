# プロジェクト構成

## 概要

digital-omikuji プロジェクトは、Remix を使用したデジタルおみくじアプリケーションです。VibeCoding と Gemini CLI を使用した開発環境を構築するためのドキュメントを提供します。

## ディレクトリ構成

```
digital-omikuji/
├── app/
│   ├── routes/
│   │   ├── index.tsx              # メインのおみくじコンポーネント
│   │   └── index.test.tsx         # テストコード
│   └── entry.client.tsx           # クライアントサイドエントリーポイント
├── documents/                     # プロジェクトドキュメント
│   ├── プロジェクト構成.md
│   ├── VibeCoding設定ガイド.md
│   ├── Gemini_CLI使用方法.md
│   ├── トラブルシューティングガイド.md
│   └── デプロイメントマニュアル.md
├── public/                        # 静的ファイル
├── .eslintrc.js                   # ESLint設定
├── .gitignore                     # Git無視ファイル
├── Dockerfile                     # Docker設定
├── docker-compose.yml             # Docker Compose設定
├── LICENSE                        # ライセンス
├── package.json                   # 依存関係
├── README.md                      # プロジェクト説明
└── remix.config.js                # Remix設定
```

## 技術スタック

- **フロントエンド**: Remix + React + TypeScript
- **スタイリング**: Tailwind CSS
- **コンテナ**: Docker + Docker Compose
- **テスト**: Jest
- **リンター**: ESLint
- **開発ツール**: VibeCoding + Gemini CLI

## 主要コンポーネント

### おみくじ機能
- 重み付け乱数による結果生成
- 「大吉」「中吉」「小吉」「凶」の4つの結果
- リアルタイム結果表示

### 開発環境
- Docker による環境統一
- VibeCoding による開発効率化
- Gemini CLI による AI 支援開発

## 設定ファイル

### package.json
プロジェクトの依存関係とスクリプトを定義

### remix.config.js
Remix フレームワークの設定

### docker-compose.yml
開発・本番環境のコンテナ設定

### .eslintrc.js
コード品質チェック設定
