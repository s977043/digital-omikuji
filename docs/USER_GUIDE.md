# デジタルおみくじ ユーザーガイド

2026年新春デジタルおみくじアプリの使い方をご紹介します。

## アプリ概要

スマートフォンを振っておみくじを引く、新年の運勢占いアプリです。

## 主な機能

### おみくじを引く

1. アプリを開く
2. 「振っておみくじを引く」画面が表示される
3. スマートフォンを振る（シェイク）
4. 抽選アニメーションが表示される
5. 運勢結果が表示される

### 運勢結果

結果画面では以下の情報が表示されます：

- **運勢**: 大吉、中吉、小吉、吉、末吉、凶、大凶
- **テーマ**: その運勢に合わせたメッセージ
- **運勢詳細**: 恋愛運、金運、仕事運などの詳細

### シェア機能

結果画面の「シェア」ボタンから、おみくじ結果をSNSでシェアできます。

- X（Twitter）
- LINE
- その他のアプリ

### 運勢手帳（履歴機能）

過去に引いたおみくじの結果を確認できます。

- ホーム画面から「履歴」をタップ
- 日付ごとの結果一覧が表示される

### 1日1回制限

おみくじは1日1回のみ引くことができます。

- 同じ日に再度アプリを開くと、その日の結果が表示されます
- 翌日になると再びおみくじを引けます

## 対応環境

- iOS 14.0以上
- Android 8.0以上
- Webブラウザ（Chrome、Safari、Firefox、Edge）

## 言語対応

- 日本語
- 英語

端末の言語設定に応じて自動的に切り替わります。

## プライバシー

- アプリはおみくじ結果をデバイス内にのみ保存します
- 外部サーバーへの送信は行いません
- 詳細は「プライバシーポリシー」画面をご確認ください

## よくある質問

### Q: おみくじが引けない

A: 以下をお試しください：

- アプリを再起動する
- 端末のセンサー（加速度計）が有効か確認する
- 十分な強さでスマートフォンを振る

### Q: 結果が保存されない

A: ストレージ容量が不足している可能性があります。端末の空き容量を確認してください。

### Q: シェアできない

A: シェア先のアプリがインストールされているか確認してください。
