# 貢献ガイド

digital-omikuji プロジェクトへの貢献に興味を持っていただき、ありがとうございます！

## 貢献方法

### 1. 開発環境の準備

```bash
# リポジトリをフォーク後、クローンしてください
git clone https://github.com/YOUR_USERNAME/digital-omikuji.git
cd digital-omikuji

# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev
```

### 2. プルリクエストの作成

1. **フォーク**: リポジトリをフォークしてください
2. **ブランチ作成**: 機能別にブランチを作成してください
   ```bash
   git checkout -b feature/new-feature
   ```
3. **コミット**: 明確なコミットメッセージを使用してください
4. **テスト**: 変更をテストしてください
   ```bash
   npm run test
   npm run build
   ```
5. **プルリクエスト**: メインリポジトリに対してプルリクエストを作成してください

### 3. コーディング規約

- **TypeScript**: 型安全なコードを書いてください
- **ESLint**: `npm run lint` でコードスタイルをチェックしてください
- **コミットメッセージ**: 変更内容を明確に記述してください

### 4. 機能追加・バグ修正

- **Issue 作成**: 新機能やバグ報告は Issue を作成してください
- **ドキュメント更新**: コード変更に伴い、ドキュメントも更新してください
- **テスト追加**: 新機能には適切なテストを追加してください

### 5. ドキュメント貢献

プロジェクトのドキュメント改善も歓迎します：

- README.md の改善
- documents/ フォルダ内のドキュメント更新
- コメントの追加・改善

## セキュリティ

セキュリティ上の問題を発見した場合は、公開 Issue ではなく、メンテナーに直接連絡してください。

## ライセンス

このプロジェクトに貢献することで、あなたの貢献が MIT ライセンスの下でライセンスされることに同意したものとみなされます。
